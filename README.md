# Gift Shop: Flask E-Commerce Application

A simple e-commerce web application built with Flask and MySQL. It supports user authentication, product browsing, cart management, and order placement. The project is deployed using Render (Flask backend) and Aiven (MySQL database).

**Live Demo:** https://gift-shop-pandead.onrender.com

---

## Features

- User registration and login  
- Category-based product browsing  
- Search functionality  
- Add to cart, update quantities, remove items  
- Order checkout and order history  
- Admin interface for adding and managing products  
- Email-based OTP for password recovery  

---

## Tech Stack

- **Backend:** Flask (Python)  
- **Database:** MySQL (hosted on Aiven)  
- **Deployment:** Render (Web Service)  
- **Email Service:** Brevo API
